# Εκκρεμότητες για ενσωμάτωση στο KANONES.md (v27)

## Νέοι κανόνες κοστολόγησης (εντολές Σταύρου 20/7)

1. **Στρογγυλοποίηση τετραγωνικών: ΠΑΝΤΑ προς τα πάνω σε ακέραιο** — ποτέ δεκαδικά στα τμ πριν τον πολλαπλασιασμό με την τιμή. Π.χ. 14,72τμ → 15τμ, 36,04τμ → 37τμ.

2. **Αυτοέλεγχος πριν από κάθε κοστολόγηση**: κάθε ποσό διασταυρώνεται με τον τιμοκατάλογο (`references/times-2026.md`) και τους κανόνες υπολογισμού (π.χ. τελάρα = +20εκ ανά διάσταση) ΠΡΙΝ γραφτεί σε long_text_1/numbers. Ο κανόνας που εφαρμόστηκε γράφεται μέσα στη γραμμή κοστολόγησης, π.χ. «37τμ (+20εκ) * 22 = 814», ώστε να ελέγχεται με μια ματιά.

## Υπενθύμιση git (Mac mini)

Το τοπικό KANONES.md **v26** του Mac mini δεν έχει γίνει push (το remote είναι ακόμα στο v25, commit e3c2b47). Σειρά ενεργειών: πρώτα `git pull` (θα κατέβει και αυτό το αρχείο), μετά push του v26, μετά ενσωμάτωση των παραπάνω ως v27.
